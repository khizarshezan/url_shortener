# URL Shortener with Analytics

## Overview
A full-stack URL shortening web application that tracks click counts, timestamps, and usage analytics for each shortened link.

## Features
- Shorten any URL instantly
- Custom title for each link
- Click tracking and analytics
- Last clicked timestamp
- Most clicked URL stat
- Delete URLs
- Data stored in MySQL database

## Tech Stack
- **Python** — Backend logic
- **Flask** — Web framework
- **MySQL** — Data storage
- **Bootstrap** — Frontend styling
